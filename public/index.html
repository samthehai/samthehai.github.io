Cấu trúc dữ liệu cơ bản: Array và Linked List

Contiguous vs Linked Data Structures Cấu trúc dữ liệu có thể được phân loại thành hai phần: contiguous (liên tục) hoặc linked (liên kết) tuỳ thuộc vào việc chúng đang dựa vào array (mảng) hay là pointer (con trỏ): Contiguously-allocated structures bao gồm nhiều block memory liên tiếp nhau, gồm có: arrays, matrices, heaps, và hash tables.
